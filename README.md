Опис предметної галузі
Тема: Інформаційна система обліку виконавців та концертних виступів на музичних фестивалях.
Мета розробки: Створення бази даних для автоматизації процесів збереження та обробки інформації про музичні заходи. Система призначена для накопичення даних про організаторів заходів, перелік фестивалів, артистів, що беруть у них участь, а також фінансових та логістичних деталей виступів (гонорари, розподіл по сценах).
Основні бізнес-правила галузі:
•	Один організатор може проводити кілька різних фестивалів, але кожен конкретний фестиваль підпорядковується лише одному організатору.
•	На одному фестивалі виступає багато артистів.
•	Один артист може брати участь у багатьох фестивалях протягом часу.
•	Факт виступу конкретного артиста на конкретному фестивалі характеризується унікальними параметрами: розміром гонорару та назвою сцени, де відбувається виступ.
